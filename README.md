## Captcha

200여개의 게시물/댓글을 제거한 후 CAPTCHA 경고창이 나타날 수 있습니다.

이때 다음과 같은 순서로 문제를 해결해주시면 됩니다.

- https://dcinside.com/ 접속 및 로그인
- 게시물이나 댓글 하나를 삭제하면 나타나는 reCAPTCHA를 해결
- 프로그램으로 돌아와 다시 Delete 실행
